# 🏙️ Smart City Brașov PWA - Frontend

This repo is meant to serve as the bridge between all members of the team taking part in the Kronsoft challenge.

## 🛠️ Tech Stack
* **Framework:** Angular (Standalone Components)
* **Styling:** SCSS (Mobile-First approach)
* **Architecture:** Core / Shared / Feature modular design

## 🚀 Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the server: `ng serve`
